# PERN Project Tracker

A full-stack project management application built with the PERN stack (PostgreSQL, Express, React, Node.js).

## Overview

PERN Project Tracker is a web-based application designed to help teams and individuals manage projects efficiently. It provides a user-friendly interface to create, organize, and track project tasks and progress.

## Tech Stack

- **Frontend**: React - Modern UI library for interactive interfaces
- **Backend**: Node.js with Express - Server-side runtime and web framework
- **Database**: PostgreSQL - Reliable relational database for data persistence
- **Language**: JavaScript - Powers both frontend and backend

## Features

- ✨ Create and manage projects
- 📋 Organize tasks within projects
- ✅ Track task completion status
- 👥 User-friendly interface
- 💾 Persistent data storage with PostgreSQL

## Installation

### Prerequisites

- Node.js (v12 or higher)
- PostgreSQL (v10 or higher)
- npm or yarn

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/aayusht200/PERN-ProjectTracker.git
   cd PERN-ProjectTracker
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure the database**
   - Create a PostgreSQL database
   - Update connection settings in the configuration file

4. **Start the application**
   ```bash
   npm start
   ```

5. **Access the application**
   - Open your browser and navigate to `http://localhost:3000`

## Usage

1. Navigate to the application homepage
2. Create a new project by providing a project name
3. Add tasks to your projects
4. Mark tasks as complete to track progress
5. Manage and update project details as needed
